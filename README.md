# WindManager

WindManager is an open-source Java application that allows the user to register and manage multiple Tibia accounts and their characters in a simple and objective table, that is connected to Tibia's website and returns characters informations (such as level, status, vocation).

It's  usefull for bot farmers have a control about which character is able to hunt based in stamina, which ones are online, bank balance, etc.

No longer need to open multiple clients or access the website to check these informations. Just use WindManager!
